# Youtube-Galaxy-MLOps
Two existing repos (a Fandom wiki scraper and a Streamlit 3D galaxy visualization) are being unified into a single production MLOps repo called `youtube-galaxy-mlops`. The end product is a public app at spookypharaoh.com with a live 3D creator galaxy and an LLM chatbot. It also features hands-on learning of 6 MLOps tools.
